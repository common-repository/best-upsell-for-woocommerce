== Installation ==

= Minimum Requirements =
<strong> Notes </strong>
> You must have an SSL certificate
> Pretty permalinks in Settings > Permalinks so that the custom endpoints are supported.
> Default permalinks will not work.

<strong> Free shipping setting you must have to enable free shipping like below </strong>

* Go to admin >> WooCommerce >> Settings >> Shipping >> Edit your shipping zone <a href="https://prnt.sc/22ww6gb">https://prnt.sc/22ww6gb</a>
* Then add Free Shipping method <a href="https://prnt.sc/22wwg7j">https://prnt.sc/22wwg7j</a>
* edit "Free shipping requires..." => A valid free shipping coupon <a href="https://prnt.sc/22wws8u">https://prnt.sc/22wws8u</a>

= How to change Currency setting from Woocommerce =
> Go to Admin >> Woocommerce >> settings >> Click on General tab : <a href="https://prnt.sc/--8OcSlOMjHh">https://prnt.sc/--8OcSlOMjHh</a>
> Scroll down and Changes Currency position : <a href="https://prnt.sc/H17OB-MmFThz">https://prnt.sc/H17OB-MmFThz</a>
> And It Will Take 5 Min to changes to Currency Position

<strong>Third party cache setting</strong>
= Wp Super Cache =
> Go to Admin >> Settings >> WP Super Cache >> Click on Advance tab : <a href="https://prnt.sc/26b2qup">https://prnt.sc/26b2qup</a>
> Scroll down and remove code from Rejected Cookies and add <strong>woocommerce_items_in_cart</strong> : <a href="https://prnt.sc/26b2skw">https://prnt.sc/26b2skw</a>

= Wp Fastest Cache =
> Go to Admin >> Wp Fastest Cache >> Click on Exclude tab >> In Exclude Cookies Add New Rule : <a href="https://prnt.sc/26b2xwv">https://prnt.sc/26b2xwv</a>
> Open Cookie DropDown And Select <strong>Has Woocommerce Items in Cart</strong> And Save It : <a href="https://prnt.sc/26b2yhi">https://prnt.sc/26b2yhi</a>
> Your Exclude Cookies Tab Look Like : <a href="https://prnt.sc/26b31w7">https://prnt.sc/26b31w7</a>

= WP-Optimize - Clean, Compress, Cache =
> Go to Admin >> WP-Optimize >> Click on Cache Tab : <a href="https://prnt.sc/UDNdRjnflQ5G">https://prnt.sc/UDNdRjnflQ5G</a>
> Inside Cache tab Go Advanced Settings Tab And add <strong>woocommerce_items_in_cart</strong> : <a href="https://prnt.sc/26b2x2y">https://prnt.sc/26b2x2y</a>

= WP Rocket =
> GO to admin >> WP Rocket : <a href="https://prnt.sc/8fR8T1OBQnJn">https://prnt.sc/8fR8T1OBQnJn</a>
> GO to File Optimization And Find Excluded CSS Files : <a href="https://prnt.sc/ujU6tPysghzA">https://prnt.sc/ujU6tPysghzA</a>
	And Add in Excluded CSS Files Add = bu.identixweb.com/icart/assets/css/icart_fronted.min.css
> And Scroll Down And Find Excluded JavaScript Files : <a href="https://prnt.sc/jyo70uh-YwS4">https://prnt.sc/jyo70uh-YwS4</a>
	And Add in Excluded CSS Files Add = bu.identixweb.com/icart/assets/js/icart.min.obf.js

= LiteSpeed Cache =
> Go to Admin >> LiteSpeed Cache >> Page Optimization >> Click on [2] Js Settings Tab >> Set This Setting : <a href="https://prnt.sc/P0_ktFxp5oCj">https://prnt.sc/P0_ktFxp5oCj</a>
> Go to [7] Tuning Tab : <a href="https://prnt.sc/58NOC7qjykhg">https://prnt.sc/58NOC7qjykhg</a>
> Add <strong>icart_fronted.min.css</strong> This Line Inside CSS Excludes Tab
> Add <strong>icart.min.obf.js</strong> This Line Inside JS Exclude Tab
> Add <strong>icart.min.obf.js</strong> This Line Inside JS Deferred Exclude Tab : <a href="https://prnt.sc/sbxZVD0m8s7r">https://prnt.sc/sbxZVD0m8s7r</a>

= Cache Enabler =
> Go to Admin >> Setting >> Cache Enabler >> Add <strong>woocommerce_items_in_cart</strong> In Cookies Tab And Save it : <a href="https://prnt.sc/26b3cwm">https://prnt.sc/26b3cwm</a>
